概率图

模型定义

HMM五元组\(\left ( X,Y,A,B,\pi \right )\)

观察序列集合\(X=\left { x{1},x{2},…,x_{n} \right }\)

状态集合
$$Y=\left { y{1},y{2},…,y{n} \right }$$
状态转移矩阵
$$A=\left { a{ij}|a{ij}=P\left ( s{t}=y{j}|s{t-1}=y{i} \right ) \right }$$
输出观察序列概率
$$B=\left { b{jk}|b{jk}=P\left ( o{t}=x{k}|o{t-1}=x{j} \right ) \right }$$
初始状态概率
$$\pi{i} =P(s{1}=y{i})$$

状态序列

$$S=\left {s{1},s{2},…,s{T}\right },s{i}\in Y$$

观察序列

$$O=\left {o{1},o{2},…,o{T}\right },o{i}\in X$$

模型假设

马尔科夫性假设

$$P(s{t}|s{1},s{2},…,s{t-1})=P(s{t}|s{t-1})$$

稳定性假设

$$a{ij}$$与$$t$$无关，即$$a{ij}=P\left ( s{t}=y{j}|s{t-1}=y{i} \right)$$

观察序列独立性假设

$$P(O)=p(o{1})p(o{2})…p(o_{T})”>$$
