HMM模型

[TOC]

图模型

模型定义

HMM五元组\(\left ( X,Y,A,B,\pi \right )\)

观察序列集合\(X=\left \{ x_{1},x_{2},…,x_{n} \right \}\)
状态集合\(Y=\left \{ y_{1},y_{2},…,y_{n} \right \}\)
状态转移矩阵\(A=\left \{ a_{ij}|a_{ij}=P\left ( s_{t}=y_{j}|s_{t-1}=y_{i} \right ) \right \}\)
输出观察序列概率\(B=\left \{ b_{jk}|b_{jk}=P\left ( o_{t}=x_{k}|o_{t-1}=x_{j} \right ) \right \}\)
初始状态概率\(\pi_{i} =P(s_{1}=y_{i})\)

状态序列

\(S=\left \{s_{1},s_{2},…,s_{T}\right \},s_{i}\in Y\)

观察序列

\(O=\left \{o_{1},o_{2},…,o_{T}\right \},o_{i}\in X\)

模型假设

马尔科夫性假设

\(P(s_{t}|s_{1},s_{2},…,s_{t-1})=P(s_{t}|s_{t-1})\)

稳定性假设

\(a_{ij}\)与\(t\)无关，即\(a_{ij}=P\left ( s_{t}=y_{j}|s_{t-1}=y_{i} \right)\)

观察序列独立性假设

\(P(O)=p(o_{1})p(o_{2})…p(o_{T})\)

三大问题

prediction

已知\(\mu=(A,B,\pi),O=\left \{o_{1},o_{2},…,o_{T}\right \},o_{i}\in X\)，求\(P(O|\mu)\)

explaining

已知\(\mu=(A,B,\pi),O=\left \{o_{1},o_{2},…,o_{T}\right \},o_{i}\in X\)，求\(S^{*}=\underset{S}{argmax} P(S|O,\mu)\)

learning

已知\(O=\left \{o_{1},o_{2},…,o_{T}\right \},o_{i}\in X\)，求\(\mu^{*}=\underset{\mu}{argmax} P(\mu|O)\)

prediction

问题复述，已知\(\mu=(A,B,\pi),O=\left \{o_{1},o_{2},…,o_{T}\right \},o_{i}\in X\)，求\(P(O|\mu)\)

  • 向前算法

    设置向前变量\(\alpha_{t}(i)=P(o_{1},o_{2},…,o_{t},s_{t}=y_{i}|\mu)\)，表示已知模型\(\mu\)，在\(t\)时刻，观察序列为\(o_{1},o_{2},…,o_{t}\)状态为\(y_{i}\)的概率，则\(P(O|\mu)=\sum_{i=1}^{n} \alpha_{T}(i)\)

  1. 初始化\(\alpha_{1}(i)=\pi_{i}b_{io_{1}}\)
  2. 迭代算法\(\alpha_{t+1}(j)=\sum_{i=1}^{n}\alpha_{t}(i)a_{ij}b_{jo_{t+1}}\)
  3. 终止\(P(O|\mu)=\sum_{i=1}^{n} \alpha_{T}(i)\)
  4. 时间复杂度\(O(N^{2}T)\)

  • 向后算法

    设置向后变量\(\beta_{t}(i)=P(o_{t+1},o_{t+2},…,o_{T}|s_{t}=y_{i},\mu)\)，表示已知模型\(\mu\)，在\(t\)时刻，观察序列为\(o_{t+1},o_{t+2},…,o_{T}\)状态为\(y_{i}\)的概率，则\(P(O|\mu)=\sum_{i=1}^{n}\beta_{1}(i) \pi_{i}b_{io_{1}}\)

  1. 初始化\(\beta_{T}(i)=1\)
  2. 迭代算法\(\beta_{t}(i)=\sum_{j=1}^{n}a_{ij}b_{jo_{t+1}}\beta_{t+1}(j)\)
  3. 终止\(P(O|\mu)=\sum_{i=1}^{n}\beta_{1}(i) \pi_{i}b_{io_{1}}\)
  4. 时间复杂度\(O(N^{2}T)\)

  • 向前向后算法

    $$
    \quad \quad P(O|\mu )=P(o_{ 1 },o_{ 2 },…,o_{ n }|\mu )\ =\sum _{ i=1 }^{ n }{ P(o_{ 1 },…,o_{ t },o_{ t+1 },…,o_{ n },{ s }_{ t=i }|\mu ) } \ =\sum _{ i=1 }^{ n }{ P(o_{ 1 },…,o_{ t },o_{ t+1 },…,o_{ n }|{ s }_{ t=i },\mu )P({ s }_{ t=i\quad }|\mu ) } \ =\sum _{ i=1 }^{ n }{ P(o_{ 1 },…,o_{ t }|{ s }_{ t=i },\mu )P(o_{ t+1 },…,o_{ n }|{ s }_{ t=i },\mu )P({ s }_{ t=i\quad }|\mu ) } \ =\sum _{ i=1 }^{ n }{ \alpha _{ t }(i)\beta _{ t }(i) }
    $$
